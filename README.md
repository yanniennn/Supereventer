# Supereventer

全能活动策划 Agent Skill。支持团建、年会、发布会、沙龙、展会、培训等通用活动，以及嘉年华类专项策划。

## 核心目标

协助用户高效、严谨地策划各类活动，确保从创意到执行的每一个环节都无懈可击。

## 功能说明

- **主题策划：** 根据用户初步想法，深入挖掘活动核心价值，确定具有吸引力的主题。
- **流程设计：** 构建逻辑严密的活动流程，包括开场、核心环节、互动及收尾。
- **进度管理：** 制定详细的时间表，明确各阶段任务及负责人，确保项目按时推进。
- **风险防控：** 预判潜在漏洞，提供针对性的应急预案，降低活动风险。

## 应用场景

- 企业年会、团建及产品发布会。
- 线上讲座、研讨会及各类社群活动。
- 个人庆典、婚礼及社交聚会。

## 设计思路

- **模块化思维：** 将复杂的活动策划拆解为四个标准化模块，降低操作难度。
- **细节导向：** 强调执行过程中的细节把控，以专业性提升活动品质。
- **灵活适配：** 支持多种类型活动的个性化定制，满足不同用户的差异化需求。

## 核心能力

- **四阶策划法**：确定主题 → 策划流程 → 安排进度 → 控制漏洞
- **嘉年华七步法**：主题 → 架构 → 典礼 → 分工 → 预算 → 物料 → 应急
- **双预算模式**：无预算时输出参考估算；有预算时聚焦流程与人员分工
- **文档导出**：方案完成后可导出为 Markdown 或 Word

## 安装

将本仓库克隆或下载后，把 `SKILL.md` 放到你所用 AI 助手的 skill / 技能目录，或直接在对话中引用该文件。

```bash
git clone https://github.com/yanniennn/Supereventer.git
```

## 使用示例

- 「200 人社区嘉年华，预算 3 万，传统文化主题」
- 「还没定预算，帮我估一下年会方案」
- 「方案生成后导出为 Word 文档」
